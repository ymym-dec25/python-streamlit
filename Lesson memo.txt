Streamlitの使い方
Visual Studio Codeで.pyファイルを作成する。
以下を入力する。

import streamlit as st
import numpy as np　（※numpyはPythonのライブラリ）
import pandas as pd（※pandasはPythonのライブラリ）

st.title('Streamlit 超入門')　（※ターミナルで実行するとWeb表示される。PS C:\Users\USER\Desktop\Python1> Streamlit run main.py）

st.write('DataFrame')　(※表を作成する）

df = pd.DataFrame({
    '1列目': [1, 2, 3, 4], （※カンマの後ろは空白入れる）
    '2列目': [10, 20, 30, 40]
})

st.dataframe(df, width=100, height=100)　（※表を表示する。st.write(df)でも表は表示されるが、幅や高さを指定できない.st.table(df)でも表示できるが、ソートなどできない。ただ表示するのみ）

番外編：列の最大値にハイライトを付ける。（列はaxis=0 行は1）
st.dataframe(df.style.highlight_max(axis=0), width=100, height=100)

※データ表示についてはStreamlitのDocsに情報が載っている。
https://docs.streamlit.io/develop/api-reference/data

st.bar_chart(df)　棒グラフ
st.line_chart(df) 折れ線グラフ
などグラフの表示も自由に選べる

df = pd.DataFrame(
    np.random.rand(100, 2)/[50, 50] + [35.69, 139.70], 
    columns=['lat', 'lon']
)
st.map(df)

緯度と経度を指定して、その付近の地図上に位置情報をマッピングできる。

st.write('Display Image')画像を表示する。
img = Image.open('Flower.jpg')　※パイソンのファイルと同じフォルダにある画像ファイルの場合
st.image(img, caption='お花', use_container_width=True)　※画面の幅に合わせる場合
